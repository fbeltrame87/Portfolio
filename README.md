- Gestor_de_Comercio
Commerce Management System
A robust desktop application designed to streamline inventory control and product management.
This project was developed as part of my professional training in the .NET ecosystem, focusing on clean architecture and relational database integration.

🚀 Key Features
Full CRUD Functionality: Create, Read, Update, and Delete products/articles with ease.

Advanced Filtering: Quick search and multi-criteria filtering (by category, brand, or price range).

Image Handling: Dynamic image loading for products with built-in error handling for broken links or missing files.

Relational Database: Optimized SQL queries to manage data integrity between products, brands, and categories.

Input Validation: Robust front-end validation to ensure data consistency and prevent application crashes.

🛠️ Technical Stack
Language: C#

Framework: .NET (WinForms)

Database: Microsoft SQL Server

Architecture: Multi-layered architecture (Presentation, Business Logic, and Data Access).

Tools: Visual Studio, Git, SQL Server Management Studio (SSMS).

⚙️ Installation & Setup
Clone the repository:

Bash

git clone https://github.com/fbeltrame87/Portfolio.git
Database Setup:

Create a CATALOGO_DB or...

Execute a CATALOGO_DB.sql file in your SQL Server instance to create the schema and seed data.

Configuration:

Open the solution in Visual Studio.

Update the Connection String in the data access layer (or App.config) with your local SQL instance credentials.

Run:

Press F5 to build and launch the application.
